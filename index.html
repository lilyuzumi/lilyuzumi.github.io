    • Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software

    GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

    Vizualiza os arquivos que foram alterados

    git status

    Prepara um arquivo para ser versionado

    git add nome_de_um_arquivo

    Gera a versão das notificações

    git comit -m "uma mensagem legal do que foi feito nos arquivos nesta versão"

    Envia para o Github

    git push origin main
